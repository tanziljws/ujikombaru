âœ¨ CSS PROPERTIES EXPLAINED:
----------------------------

**display: flex**
- Enables flexbox layout
- Children become flex items

**flex-direction: column**
- Stack items vertically
- Image on top, body below

**flex-wrap: wrap**
- Allow items to wrap to next row
- Essential for responsive grid

**height: 100%**
- Stretch card to fill column height
- All cards in same row have equal height

**flex: 1**
- Grow to fill available space
- Shrink if needed
- Flexible sizing

**flex-shrink: 0**
- Prevent element from shrinking
- Maintain fixed dimensions
